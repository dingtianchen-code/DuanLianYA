# 锻炼YA Design System

Version: 1.0


# 1. 设计理念

锻炼YA的视觉核心：

自然、生长、陪伴。


设计目标：

让用户打开App时，
感受到：

- 温暖
- 安心
- 积极
- 长期成长


锻炼YA不是竞技型健身产品。

不是强调：

- 比别人更强
- 更快达到目标

而强调：

每天照顾身体一点点。



---

# 2. 核心视觉符号


## 身体之花 Flower

身体之花是首页核心视觉。


含义：

不同身体区域的训练积累，
最终形成整体成长。


花不是：

成绩排名。

而是：

身体觉察和长期坚持的可视化。



---

# 3. 色彩系统


整体：

自然色系。


关键词：

- 春天
- 夏天
- 秋天
- 冬天


颜色根据季节变化。

避免：

高刺激健身App风格。


不使用：

强烈警告红色

过度竞技色彩



---

# 4. 动画原则


动画目标：

鼓励，而不是评价。


训练完成后：

播放2-3秒动画。


例如：

腰背训练完成：

1. 卡通人物弯腰
2. 腰部出现不适提示
3. 提示逐渐消失
4. 身体恢复挺直
5. 人物微笑并点赞


动画表达：

“你今天照顾了自己的身体。”


不是：

“你的疾病被治愈。”



---

# 5. UI风格


关键词：

- 简洁
- 柔和
- 有生命感
- 陪伴感


避免：

传统健身App：

- 大数字
- 卡路里竞争
- 排名
- 强目标压力



---

# 6. 组件原则


所有组件保持：

- 圆角
- 呼吸感
- 轻量动画


核心组件：

FlowerView

WorkoutCard

GrowthCard

AnimationView



---

# 7. SwiftUI实现原则


视觉组件独立。


例如：

FlowerView

不应该包含：

用户数据逻辑。


数据由：

ViewModel

提供。


保持：

UI
↓

ViewModel
↓

Model
