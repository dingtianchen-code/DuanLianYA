# Front Production Release Checklist

- [ ] Front source vectors were manually calibrated against Master Reference V3.
- [ ] Neutral closed beak has no new seam, horizontal line, or contour.
- [ ] Master Overlay and difference image approved.
- [ ] Hidden Geometry Register entries used by Front are approved.
- [ ] Layer names and draw order match CharacterSystem v1.1.
- [ ] Pivot Overlay approved.
- [ ] SVG import into Rive preserves appearance, clipping, gradient, hierarchy, and names.
- [ ] Rig extreme poses show no cracks, exposed hidden geometry, or pivot drift.
- [ ] Idle approved at 0.25x and 1x.
- [ ] Blink uses dedicated Blink Pose and is approved at 0.25x and 1x.
- [ ] Wave is driven primarily from the shoulder pivot and is approved at 0.25x and 1x.
- [ ] Reduced Motion behavior approved.
- [ ] No generated or auto-traced output was promoted without manual path calibration.
- [ ] Approval Register updated before release.
